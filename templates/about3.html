Arithmetic Logic Unit (ALU) and Multiplexer (MUX)

An arithmetic logic unit (ALU) is a major component of the CPU computer system. It does all processes related to arithmetic and logic operations that need to be done on instruction words and registers. Therefore, the ALU is like the calculator of the CPU.

A multiplexer (MUX) is a device that can receive multiple input signals and synthesize a single output signal in a recoverable manner for each input signal. It is also an integrated system that usually contains a certain number of data inputs and a single output. The Mux is an important part of the CPU, since it allows it to execute a variety of instructions that need to be computed and executed differently.
